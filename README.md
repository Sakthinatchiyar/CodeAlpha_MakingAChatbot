

## Project Overview

This project is developed as part of the CodeAlpha Cloud Computing Internship.

The objective of this project is to create an AI-powered chatbot that can provide instant responses to user queries through a website interface. The chatbot uses a retrieval-based approach with predefined response patterns.

## Features

- Instant response to user queries
- Retrieval-based chatbot model
- User-friendly web interface
- Predefined question and answer patterns
- Lightweight and easy to deploy
- Built using Python and Flask

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript

## How It Works

1. User enters a message in the chatbot interface.
2. Flask receives the message.
3. The chatbot checks predefined responses.
4. A matching response is returned instantly.
5. The response is displayed on the webpage.

## Installation

### Clone Repository

```bash
git clone https://github.com/YourUsername/CodeAlpha_MakingAChatbot.git
```

### Navigate to Project Folder

```bash
cd CodeAlpha_MakingAChatbot
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

### Open Browser

```text
http://127.0.0.1:5000
```
